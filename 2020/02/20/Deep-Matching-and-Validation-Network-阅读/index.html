摘要

拼接检测和定位算法通常是输入图像，产生二值决策是否是篡改图像，并且分割出拼接区域。目前算法的缺点1.手工特征对压缩这类操作鲁棒性差。2.（定位和检测）每一阶段，各自优化。
本文提出DMVN网络解决CISD（约束图像拼接和检测）问题，能够同时定位和检测。向DMVN输入原图和篡改后的图，DMVN评估原图被篡改的可能性并获取两幅图的mask

网络结构

CISD问题定义为给定图P Q,确定Q是否是拼接篡改，并找出P Q中相对应的拼接区域
整体网络结构如图3a：
CNN Feature Extractor提取特征—-Deep Dense Matching module输入图像的密集匹配—-inception-based Mask Deconvolution module分割mask—-Visual Consistency Validator module一致性验证—-siameselike module概率估计

Mask Deconvolution module
逐步将响应图解卷积2倍，直到其大小达到输入的大小，即256 * 256。在每个反卷积阶段，我们应用一个初始模块，该模块具有更大的滤波器尺寸和更少的滤波器，生成Pm Qm

拼接检测

首先使用Visual Attention模块将CNN功能中的所有非拼接区域归零。再使用深度特征提取器提取特征，计算两者的差异，推断P是Q的供体的概率。

实验

训练

依据44描述的非监督生成过程，使用SUN2012 object detection
dataset 和 MS COCO dataset创建训练数据集。把x的对象变换（缩放旋转平移随机光照变化）粘贴到Y得到Z,XY是正样本,Z是负样本。得到150万，30万，30万合成样本用于训练，验证和测试。
使用the deep Keras learning framework with the theano backend框架
首先只关注DMVN模型的定位分支。一旦该定位分支收敛，我们将冻结其权重，添加到检测分支上，并训练检测分支直到收敛。最后，我们解冻所有DMVN权重，并使用随机梯度优化器以学习速率1e-5和动量0.9端对端地训练整个模型。
在我们的综合训练，验证和测试数据集上，我们在定位分支上分别达到98.52％，98.67％和97.88％的预测准确度，在检测分支上分别达到97.75％，97.53％和97.69％的预测准确度。
