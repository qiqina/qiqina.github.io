BusterNet: Detecting Copy-Move Image Forgery with Source/Target Localization阅读

发表于 2020-02-23

论文题目 BusterNet：使用源/目标定位检测复制粘贴图像篡改
年份出处 2018 ECCV
作者 Yue Wu，Wael Abd-Almageed，Prem Natarajan 美国加州信息科学研究所
数据集
训练数据集 把MIT SUN 2012和微软COCO数据集合成出一个新数据集（使用IEEE IFS-TC First Image Forensics Challenge5 and the Wild Web tampered image dataset训练mini-det分支）
测试数据集 CASIA TIDEv2.0 dataset、CoMoFoD
实验配置
Speed measurement is based on an Intel Xeon CPU E52695@2.4GHz with a single Nivdia Titan-X GPU.
评价指标
precision recall f-score AOC分数

摘要

本文提出复制粘贴检测的CMFD结构，有一个双分支结构+融合模块。两个分支分别通过视觉伪像定位潜在的操作区域，通过视觉相似性定位复制粘贴区域。
1.是端到端，可以直接进行优化。2.不仅检测出篡改，还可生成源和目标的mask

网络结构

整体结构如图2


虚线部分只在训练时激活。输出结果的蓝色表示原图，绿色表示复制源，红色表示复制目标。
Mani-Det分支检测操纵区域（二进制操作mask），Simi-Det分支则用于检测克隆区域（二进制复制粘贴mask），最后使用Fusion中的这两种特征来预测像素级的复制移动蒙msk，从而区分原始，源复制和目标复制。
假设输入256×256×3（也能处理其他大小图像）

操作检测分支

mani-det分割操作区域。输入图像X，使用CNN Feature Extractor提取特征，使用Mask Decoder将特征图上采样到原始图像尺寸，应用Binary Classifier来完成辅助任务。

相似性检测分支

simi-det。使用CNN Feature Extractor提取特征，通过Self-Correlation模块计算特征相似度，通过Percentile Pooling收集有用的统计信息，使用Mask Decoder将特征图上采样到原始图像大小，并应用Binary分类器完成辅助任务。
CNN Feature Extractor结构相同，但不共享权重

busternet融合

将两个分支的Mask Decoder结果输入，拼接，使用BN-Inception融合特征，预测三级CMFD mask

实验
训练

训练数据集把MIT SUN 2012和微软COCO数据集合成出一个新数据集用于训练 。（使用IEEE
IFS-TC First Image Forensics Challenge5 and the Wild Web tampered image
dataset训练mini-det分支）如图4

最终收集了10万左右图像，按8：1：1进行训练验证和测试。
步骤
stage-wise策略（i）独立训练每个分支机构的辅助任务；（ii）冻结两个分支并训练融合模块；（iii）解冻整个网络，并端到端微调BusterNet。
